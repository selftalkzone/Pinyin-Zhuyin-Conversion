About Chinese Characters' Mandarin Pronunciation/Tone Table

<Chinese Version 中文版本>


(1) As the same Chinese character can be pronounced by using different dialects such as Hokkien, Mandarin, Cantonese, Chao Zhou dialect, etc. therefore the Pronunciations/tones listed in this table are only for the Chinese Characters' Mandarin Pinyin (and Zhuyin) Pronunciation.


(2) This Pronunciation/Tone Table is adapted and edited from On-line Chinese Tools' pychart ↗.


(3) There are a total of about 400 plus number of, Chinese Characters' Mandarin Pronunciation (without tone mark), formed by various combinations of pinyin "a" to pinyin "z". For the conveniences of computer program operation, each possible pronunciation will be added with all tone marks, for example, 好=(hao/hao1/hao2/hao3/hao4/hao5)=6 tone mark for (hao), where both (hao/hao5) represent light sound tone. Further example, 对=(dui/dui1/dui2/dui3/dui4/dui5)=6 tone mark for (dui) (Ref: Table-1). Of course, (dui/dui2/dui3/dui5) may not have the corresponding Chinese characters to represent these Mandarin pronunciations, but this table will still list them together with (dui1/dui4). Thus, this table will contain more pronunciation tones that are not used than the normal pronunciation table.

NB1: The Pinyin Pronunciation Table use here contains 436 pronunciations (without tone mark). And after tone marks are added, it has a total of 436 x 6 = 2616 pronunciations (with tone mark).

NB2: Of the 436 pronunciations (without tone mark), some of them are the same pronunciations but with different pinyin symbols to represent them, therefore they are repeated pronunciations, for example, in nü=nu:=nv=nyu, nü can be represented by nu:, nv, or by nyu, hence the same pronunciation of nü, is also the pronunciation of nu: nv, and nyu i.e. becomes 4 repeated pronunciations.

NB3: Of the 436 pronunciations (without tone mark), there are 8 groups with repeated pronunciation: (1) ê=e^=2Repeats, (2) lü=lu:=lv=lyu=4Repeats, (3) lüe=lu:e=lve=lyue=4Repeats, (4) m=m=2Repeats, (5) n=n=2Repeats, (6) ng=ng=2Repeats, (7) nü=nu:=nv=nyu=4Repeats, (8) nüe=nu:e=nve=nyue=4Repeats. Hence the 8 groups of pronunciations, combine with their 16 repeated pronunciations, becomes 24 pronunciations.

NB4: Of the 436 pronunciations (without tone mark), there are 6 groups of less-used pronunciations: (1) ê, (2) hm, (3) hng, (4) m, (5) n, (6) ng.

NB5: Of the 436 pronunciations (without tone mark), if the 16 repeated pronunciations are not counted, the actual number of pronunciations = 436-16 = 420. If the less-used pronunciations are also not counted, the actual number of (frequent-used) pronunciations = 436-16-6 = 414.


(4) In most dictionary, there are pin yin ê, hm, hng, m, n (using ḿm̀ńňǹ) and ng, hence, they are included in this Pronunciation/Tone Table. However, please take note of the followings:

(a) Pin Yin problem: Last time, as the pin yin symbols (ḿm̀ńňǹ) can not be displayed properly in text file format even if they are saved in utf-8 unicode, thus, we use normal m, n to represent them (NB: The 1234 tones of pin yin for m and n will be borrowed from zhu yin's 4 tones ˉˊˇˋ, the light tone for pin yin, if needed, will not be borrowed from zhu yin's ˙, it will use · for pin yin). Now, since ḿm̀ńňǹ can be displayed properly, we will add and include them into the Pronunciation/Tone Table (but m̀ is not included eventually because Javascript can't correctly process it). For pin yin symbol ê, as there is no display problem, we will use it directly.

Furthermore, the 1234 tones of pin yin for ê, hm, hng and ng will also be borrowed from zhu yin's 4 tones ˉˊˇˋ, the light tone for pin yin, if needed, will not be borrowed from zhu yin's ˙, it will use · for pin yin.

(b) Zhu Yin problem: In pin yin, hng and ng use ng, but ng in zhu yin is an old zhu yin sound that use 兀 as zhu yin tone, the modern zhu yin no longer has 兀 as tone, so we have to use the Chinese character 兀 to represent it.

(c) In Pin Yin, sometimes er can be written as r, for example, "花儿" in Pin Yin can be "huar", however, only er is recorded in this Table, r will not be recorded.

(6) Appendix: The followings are the way where Pin Yin (PY), Zhu Yin (ZY) and their tone marks are arranged in dictionary books (using pin yin of "ma" as example)


S/No PYnum PY ZY Tone 声调 Remark 注解

2 ma2 má ㄇㄚˊ 2nd tone
3 ma3 mǎ ㄇㄚˇ 3rd tone
4 ma4 mà ㄇㄚˋ 4th tone
5 ma5 ·ma ˙ㄇㄚ Light tone (Pinyin Light tone · ) and (Zhuyin Light tone ˙ ) default mark left

Note: In dictionary books, the pin yin light tone will always has · mark on the left. Therefore there is only "·ma" light tone in dictionary books, no "ma" light tone. But in our Tone Table usage (for computer), there will be cases of ·ma=ma, or even the possiblity of ·ma=ma=ma0=ma5 when we input pin yin using computer keyboard. So please take note that in computer usage, the same pin yin light tone can have a few different written forms (eg: ·ma, ma, ma0, or ma5).
