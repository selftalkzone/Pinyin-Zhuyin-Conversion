如何引用音节表方法 (英文说明)
How To Adapt and Use Tone Table Method

Step-1: Download a copy of external javascript file pyzyToneTableMtdNew.js which contains convertPyZy() and convertZyPy() methods. Create a html file and link to the js file via < script src="pyzyToneTableMtdNew.js" >< /script >.


convertPyZy() method will receive pyTxt (eg: ni3 hao3 ma) as parameter and return an object with three properties, i.e: pyNumResult, pyResult and zyResult. If fail to convert, it will return the original pyTxt with ??? for all the three object properties.

convertZyPy() method will receive zyTxt (eg: ㄏㄠˇ ㄐㄧㄡˇ ㄅㄨˋ ㄐㄧㄢˋ) as parameter and return an object with three properties, i.e: pyNumResult, pyResult and zyResult. If fail to convert, it will return the original zyTxt with ??? for all the three object properties.

Note: When the external javascript file pyzyToneTableMtdNew.js is loaded with the html file, it will prompt user to add extra html codes, user can decide add or not add accordingly.


Step-2: Create javascript functions, says: pyToZy() and zyToPy() (as shown below), to use convertPyZy() and convertZyPy() methods.

Step-3: Create html text input and buttons to activate the pyToZy() and zyToPy() functions.
